adult-offence-region-nz
=======================

This data is originally download from dataLab. using dc.js, crossfilter, bootsrap and topojson to display the multiple dimension data. Some regions do not have data, because the region in this map is bit different from court cluster region area. we need the court cluster region shape file to coreect this later. 

How to run it? under your folder with index.html

python -m SimpleHTTPServer 9999
